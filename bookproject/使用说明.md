### 软件安装说明

#### 下载软件包

1.浏览器输入：https://github.com/geeeeeeeek/book 点击下载zip包，然后将zip包复制到服务器的C盘目录下。
2.在C盘根目录新建文件夹，命名为book，将zip包解压到book文件夹中。
3.鼠标右键点击book，点击属性，修改该文件夹权限为可读写。



#### 配置数据库

1.安装MySQL数据库软件

在服务器的浏览器输入：https://downloads.mysql.com/archives/installer/
打开MySQL官方网站，下载MySQL 5.7.20版本到本地，然后双击msi将该数据库软件一步一步安装到服务器。

2.创建数据库
在cmd窗口，输入：
CREATE DATABASE IF NOT EXISTS book DEFAULT CHARSET utf8 COLLATE utf8_general_ci;


#### 配置服务

1.安装python

浏览器地址栏输入：https://www.python.org/downloads/ 选择python 3.8.12版进行下载，下载成功后，
双击exe，按照说明一步一步的安装到本地。

2.安装依赖包

打开cmd命令行窗口，通过cd命令定位到C盘的book/bookproject目录下，输入：pip install -r requirements.txt 安装依赖包

安装成功后，看到如下提示：
xxxx

3.迁移数据库表

在cmd命令行窗口，输入下面的命令，完成数据库表的创建。
```
python manage.py makemigrations;

python manage.py migrate;

python manage.py makemigrations myapp;

python manage.py migrate myapp;
```


4.在cmd命令行窗口，输入: python manage.py runserver 启动服务。
服务启动后，界面如下：
xxxx

#### 配置html服务

1.安装nginx软件
浏览器输入：https://nginx.org/en/download.html 下载nginx/Windows-1.20.2版本软件，
保存zip包到服务器的C盘目录下，并将该zip包解压到当前目录。


2.安装node
浏览器地址栏输入：http://nodejs.cn/download/ 点击windows安装包，将安装包下载到本地，
然后双击安装包按照说明一步一步进行安装，安装完毕后，打开cmd命令行窗口，输入：
node -v 即可打印出node版本号，表示安装成功。界面如下：
xxxx

3.安装依赖
在cmd命令行窗口，通过cd命令定位到C盘的book/web目录下，输入：
```
npm install 
```
安装依赖

4.构建html包
在cmd命令行窗口，输入：npm build 构建html包，构建成功后，界面如下：
xxxx

并在当前文件夹自动生成dist文件夹，下面将dist文件夹复制到C盘目录下

5.修改nginx配置文件
修改nginx目录的的conf文件夹下的nginx.conf文件，用记事本打开该文件，添加：
```
server {
    listen 80; 
    server_name  localhost;
    location / {
      root   html;              
      index  index.html index.htm; 
    }
}
```
然后Ctrl+S保存文件，关闭即可。


6.启动nginx

在cmd窗口，进入到nginx/bin目录下，输入: start nginx 启动nginx服务

8.检查配置

浏览器地址栏输入: http://localhost:80/index 显示界面如下，表示安装成功
xxxx

# 功能说明

整个软件包括前台和后台两个部分。
前台功能包括：首页图书列表、图书详情页、图书的借还、用户中心模块。
后台功能包括：总览、借阅管理、图书管理、分类管理、标签管理、评论管理、用户管理、运营管理、日志管理、系统信息模块。
